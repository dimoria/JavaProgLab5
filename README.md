# Лабораторна робота №5  
## Тема: Наслідування та поліморфізм у Java

### Виконала:
**Дубенок Діана**, група **ІМ-34**  
Варіант: **4**
## Опис
Програма реалізує ієрархію музичних композицій, що складається з узагальненого класу `MusicComposition` та трьох нащадків: `PopSong`, `RockSong`, `JazzSong`.
Реалізовано:
- створення альбому (масиву композицій);
- обчислення загальної тривалості альбому;
- сортування композицій за стилем;
- пошук композиції за діапазоном тривалості;
- обробку основних винятків;
- збереження та завантаження альбому з текстового файлу.
